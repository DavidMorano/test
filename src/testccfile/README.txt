YESYCCFILE

This tests the CCFILE object. Optionally, the USTREAM object can
be tested, both are optionally tested in 'text' (read file line-by-line)
mode.

Synopsis:
$ testccfile.x <type(s)> <file(s)>

Arguments:
<type(s)>	One of:
			ccfile		- use CCFILE object
			stream		- use USTREAM object
			text		- use text mode if available
